# AutoJump chrome 网页外链跳转插件


- 适用于 知乎, CSDN ,掘金,微信开放社区, gitee.com, oschina 点击链接后直接打开 避免手工再次点击 "继续访问" 能快一秒是一秒

- 新增: angular.io 自动跳转到中文站 angular.cn
- 新增: github 仓库加速地址
